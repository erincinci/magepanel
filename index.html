MagePanel

Web-based user interface for Magallanes Deployment Tool.

Enjoy!

(Requires Node v0.10 or newer)

Recommended Installation

>> git clone https://github.com/erincinci/magepanel.git MagePanel
>> cd MagePanel
>> sudo npm install pm2 -g
>> npm start

Prerequisites

  • For Debian Based Linux (Ubuntu, ...):
>> sudo apt-get install php5-cli
  • For RedHat Based Linux (CentOS, ...):
>> sudo yum install php-cli

## MagePanel ##
![Alt text](http://s27.postimg.org/k2ek5zrjn/magepanel.png)

This project is a web-based user interface for [Magallanes Project](http://magephp.com/).

Enjoy!

*(Requires [Node](http://nodejs.org/) v0.10.2 or newer)*
